# 🧠 AI SYSTEM INSTRUCTIONS FOR STREAM-SDK
1. **Teknoloji:** TypeScript, WebAudio API, Protobuf.
2. **Smart Media, Dumb Logic:** SDK asla iş mantığı (AI prompt, routing, veritabanı kararları) barındırmaz. Ancak ağ bant genişliğini korumak için **Medya Zekası (VAD, Jitter Buffering, Noise Gate)** barındırmak ZORUNDADIR. Sunucuya sessizlik yollanması yasaktır.
3. **Zero-Latency Barge-in:** Kullanıcı konuştuğu an (VAD), SDK hoparlör tamponunu anında temizlemeli ve sunucuya mikrosaniyeler içinde `EVENT_TYPE_INTERRUPT` sinyalini iletmelidir.
4. **State Persistence Yasağı:** Tarayıcı hafızasında (LocalStorage vb.) kalıcı oturum, şifre veya hassas veri tutulamaz.
